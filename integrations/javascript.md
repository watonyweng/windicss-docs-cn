# Windi CSS 的 JavaScript API {#windi-css-javascript-api}

<PackageInfo name="windicss" author="voorjaar" />

## 关于 {#about}

如果使用命令行工具不能适配你的工作流，你可以直接使用 Windi CSS API。

## 安装 {#install}

添加包:

```bash
npm i -D windicss
```

## 用法 {#usage}
